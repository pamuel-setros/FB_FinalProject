# Images Folder - Cleveland Browns Dashboard

This folder is reserved for Cleveland Browns branding assets and imagery used on the 12% Rule Dashboard.

## Recommended Image Files

You can add the following images to enhance the dashboard's Browns theming:

### Logo Files
- `browns-logo.png` - Official Cleveland Browns logo
- `browns-wordmark.png` - Cleveland Browns wordmark/text logo
- `browns-helmet.png` - Browns helmet icon

### Background & Icons
- `bg-texture.png` - Optional texture background for headers/footers
- `nav-icon.png` - Navigation or menu icon

## Color Reference

**Official Cleveland Browns Colors:**
- **Primary Brown:** #311C15
- **Orange:** #FF6600
- **White:** #FFFFFF
- **Steel Blue (Secondary):** #003D7A

## Usage Notes

- Images should be optimized PNG or SVG format for web use
- Recommended dimensions:
  - Logos: 200-400px width
  - Icons: 32-64px
  - Backgrounds: 1200px+ width
- All images are served via the same directory structure as `index.html`

## Legal Notice

Only use Cleveland Browns imagery that you have explicit rights to use. Consult official NFL/Browns media resources for properly licensed assets.
